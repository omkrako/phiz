## איך לצלם screenshot במכשיר Android?

### באמולטור Android Studio:
1. הפעילו את האפליקציה באמולטור
2. נווטו למסך הרצוי
3. לחצו על סמל המצלמה בסרגל הכלים של האמולטור
4. התמונה תישמר אוטומטית

### במכשיר פיזי:
- **Samsung:** Power + Volume Down
- **Google Pixel:** Power + Volume Down
- **OnePlus:** Power + Volume Down
- **Xiaomi:** Power + Volume Down

## הוספת התמונות למדריך

1. צלמו את כל תמונות המסך הנדרשות
2. שמרו אותן בשמות הקבצים המצוינים למעלה
3. העתיקו את הקבצים לתיקייה זו
4. ודאו שהתמונות בפורמט PNG או JPG
5. גודל מומלץ: 1080x1920 (רזולוציית טלפון סטנדרטית)

## טיפים לתמונות איכותיות

- השתמשו במכשיר עם מסך ברור ונקי
- ודאו שאין תוכן רגיש (כתובות אימייל אמיתיות וכו')
- השתמשו בנתונים לדוגמה בלבד
- התמונות צריכות להיות ברורות וקריאות
- הימנעו מצילומי מסך עם הודעות שגיאה (אלא אם כן זה חלק מהתיעוד)
