# הסבר הסרטון - אפליקציית Physics Simulator

## סקירה כללית
האפליקציה היא כלי לימודי לפיזיקה המאפשר לתלמידים ללמוד את חוקי ניוטון באמצעות סימולציות אינטראקטיביות ומבחנים.

---

## מסך 1: מסך בית התלמיד (StudentHomeActivity)

### תפקיד המסך
מסך נחיתה ראשי המופיע לתלמיד מיד לאחר התחברות מוצלחת למערכת.

### אלמנטים במסך
| אלמנט | תיאור |
|--------|--------|
| שורת ברכה | מציגה "Welcome, [שם התלמיד]" באופן אישי |
| תצוגת ניקוד | מציגה את סך הנקודות שהתלמיד צבר במבחנים |
| כרטיסיית Physics Simulation | כניסה לסימולציית הפיזיקה האינטראקטיבית |
| כרטיסיית Take Quiz | כניסה למבחן בנושא חוקי ניוטון |
| כרטיסיית View Grades | צפייה בהיסטוריית הציונים |
| כפתור Logout | התנתקות מהמערכת וחזרה למסך ההתחברות |

### פעולות עיקריות
1. **טעינת נתוני משתמש** - קריאה מ-Firebase Firestore להצגת שם וניקוד
2. **ניווט למסכים** - לחיצה על כרטיסייה מעבירה למסך המתאים
3. **התנתקות** - יציאה מהחשבון וניקוי הסשן

### קוד רלוונטי
- קובץ: `StudentHomeActivity.java`
- Layout: `activity_student_home.xml`

---

## מסך 2: סימולציית פיזיקה (PhysicsSimulationActivity)

### תפקיד המסך
כלי לימוד אינטראקטיבי המדגים את חוקי ניוטון באמצעות סימולציה ויזואלית של גוף בתנועה.

### אלמנטים במסך
| אלמנט | תיאור |
|--------|--------|
| אזור סימולציה | תצוגה גרפית של אובייקט נע על משטח |
| סליידר Mass | שליטה במסת הגוף (0.1 - 10 ק"ג) |
| סליידר Force | שליטה בכוח המופעל (0 - 100 ניוטון) |
| סליידר Friction | שליטה במקדם החיכוך (0 - 1) |
| סליידר Angle | שליטה בזווית המשטח (0° - 90°) |
| תצוגת Acceleration | הצגת התאוצה בזמן אמת (m/s²) |
| תצוגת Velocity | הצגת המהירות בזמן אמת (m/s) |
| תצוגת Position | הצגת המיקום בזמן אמת (m) |
| הסבר חוקי ניוטון | טקסט המסביר את שלושת החוקים |
| כפתור Start | הפעלת הסימולציה |
| כפתור Reset | איפוס הסימולציה למצב התחלתי |
| כפתור Back | חזרה למסך הבית |

### פעולות עיקריות
1. **שינוי פרמטרים** - הזזת סליידרים משנה את תנאי הניסוי
2. **הפעלת סימולציה** - לחיצה על Start מתחילה אנימציה לפי הנוסחה F=ma
3. **צפייה בתוצאות** - התאוצה, מהירות ומיקום מתעדכנים בזמן אמת
4. **איפוס** - החזרת הגוף למיקום התחלתי

### עקרונות פיזיקליים מודגמים
- **חוק ניוטון הראשון**: גוף נשאר במנוחה או בתנועה קבועה אלא אם פועל עליו כוח
- **חוק ניוטון השני**: F = ma (כוח שווה מסה כפול תאוצה)
- **חוק ניוטון השלישי**: לכל פעולה יש תגובה שווה והפוכה

### קוד רלוונטי
- קובץ: `PhysicsSimulationActivity.java`
- View מותאם: `PhysicsSimulationView.java`
- Layout: `activity_physics_simulation.xml`

## ארכיטקטורה טכנית

### טכנולוגיות בשימוש
- **שפה**: Java
- **פלטפורמה**: Android SDK
- **בסיס נתונים**: Firebase Firestore
- **אימות**: Firebase Authentication
- **עיצוב**: Material Design Components

### מבנה הפרויקט
```
app/src/main/java/com/example/phiz/
├── MainActivity.java          # נקודת כניסה
├── LoginActivity.java         # מסך התחברות
├── RegisterActivity.java      # מסך הרשמה
├── StudentHomeActivity.java   # מסך בית תלמיד
├── TeacherHomeActivity.java   # מסך בית מורה
├── PhysicsSimulationActivity.java  # סימולציה
├── PhysicsSimulationView.java      # View מותאם לסימולציה
├── QuizActivity.java          # מסך מבחן
├── GradesActivity.java        # מסך ציונים
└── User.java                  # מודל משתמש
```

## סיכום
האפליקציה מספקת חוויית למידה אינטראקטיבית לפיזיקה, עם דגש על הבנת חוקי ניוטון דרך ניסוי וירטואלי. המשתמש יכול לשנות פרמטרים ולראות בזמן אמת כיצד הם משפיעים על תנועת הגוף.
